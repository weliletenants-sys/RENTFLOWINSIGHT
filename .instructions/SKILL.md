This skill guides the creation of distinctive, professional frontend interfaces specifically for the **RentLedger SaaS Tenant Management System**.

The system is a **financial-style SaaS platform** used by estate companies to manage properties, tenants, leases, and payments.

All UI must feel **professional, trustworthy, and enterprise-grade**.

The design must avoid generic AI aesthetics and instead produce **purposeful, production-ready UI** suitable for real SaaS software.

# Technology Stack

All UI must follow the RentLedger stack:

Backend
- Laravel

Frontend
- Blade templates
- Livewire components
- Alpine.js interactions

Architecture
- Modular system using **nWidart Laravel Modules**

Styling
- TailwindCSS or structured CSS variables

Interactive elements
- Livewire + Alpine.js

DO NOT generate:

- React components
- Vue applications
- Angular apps
- standalone HTML prototypes

All interfaces must integrate naturally into the **Laravel Blade + Livewire ecosystem**.
